## Git Convention

커밋헤더 : 어떤 기능인지 설명 #지라이슈번호

상세설명 순으로 작성

ex) </br>
feat : 로그인기능 구현 #00

jwt 토큰을 활용해 로그인 인증 기능 구현

### 커밋 헤더

| 커밋 헤더    | 	설명                |
|----------|--------------------|
| feat     | 새로운 기능 구현          |
| refactor | 기존 기능 로직 변경        |
| chore    | 환경설정, 주석제거 등       |
| fix      | 버그 수정              |
| style    | 코드 스타일 변경          |
| docs     | 문서 정리              |
| test     | 테스트 코드             |
